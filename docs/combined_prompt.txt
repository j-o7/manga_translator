Yk what? New idea and script

Script takes japanese base images and gt base images directory as args
Both folders get preprocessed into the 8 techniques using preprocess.py. Hence 16 images saved under outputs/pg_{no} as:
jap_{tech}.png and gt_{tech}.png
Each image goes through OCR:
jap using pp_ocrv5_jap.py, gt using pp_ocrv5_en.py
The base OCR image is saved as:
jap_{tech}_base_ocr.png and gt_{tech}_base_ocr.png
The merged OCR image is saved as:
jap_{tech}_merged_ocr.png and gt_{tech}_merged_ocr.png
The merged JSONs files are also stored as:
jap_{tech}_merged.json and gt_{tech}_merged.json
Then the japanese OCR json file is used for translation using translate_jp_to_en.py
A new json file is created with all the paths and the following info:
input_img path
bounding boxes from jap_{tech}_merged_ocr.json
gt_texts
translation_{tech}_fb
translation_{tech}_hl
translation_{tech}_nllb
gt_scores
